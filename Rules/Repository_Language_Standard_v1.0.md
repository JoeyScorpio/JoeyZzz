# Repository Language Standard v1.0

## Purpose

本文件定义本仓库所有文档与命名的语言使用规范，确保一致性与可维护性。

## Core Rule

- 管理、决策、知识类文档：**中文优先**。
- 自动化与 AI 协作指令文档：**英文优先**。
- 命名（仓库、文件夹、文件）：**英文优先**。
- 双语写作**不是默认要求**。

## File-Level Policy

| 类别 | 示例文件 | 语言 |
|------|---------|------|
| 项目文档、研究笔记、决策文档 | README.md, index.md, BOOT.md | 中文优先 |
| 自动化 / AI 协作指令 | CLAUDE.md, AGENTS.md | 英文优先 |

## Naming Policy

仓库名、文件夹名、文件名统一使用**英文**。

## Bilingual Exception

完整双语写作仅允许用于以下场景：

1. 面向公众的入口页面（public-facing entry pages）
2. 明确指定的协作页面（explicitly designated collaboration pages）

除上述情况外，不进行双语写作。

## Authority

本文件是本仓库语言规范的**唯一权威来源**（single source of truth）。

## Update Rule

任何语言规范的变更必须通过修改本文件完成，不得在其他文件中另行定义。
